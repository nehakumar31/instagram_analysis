1. Download Neo4j Community edition for windows
2. At the location: neo4j-community-4.1.11-windows\neo4j-community-4.1.11\bin , execute command : neo4j.bat console to 
start a local Neo4j server.
3. A server instance will run and can be interacted using local browser: http://localhost:7474/browser/
4. Package dependency:
!pip install py2neo
